96 !* FNDBIT *
97 !* *
98 !* This subroutine computes the number of packing bits given the *
99 !* maximum number of significant digits to preserve or the binary *
100 !* precision to store the data. The binary precision is given as a *
101 !* negative integer, ISCALE will always be zero in this case. *
102 !* *
103 !* The binary precision translates as follows: *
104 !* -1 => store data to nearest 1/2 *
105 !* -2 => store data to nearest 1/4 *
106 !* -3 => store data to nearest 1/8 *
107 !* *
108 !* Note that a fractional number of significant digits is allowed. *
109 !* *
110 !* FNDBIT ( AMIN, AMAX, RDB, NBITS, ISCALE, IRET ) *
111 !* *
112 !* Input parameters: *
113 !* AMIN REAL Minimum value *
114 !* AMAX REAL Maximum value *
115 !* RDB REAL Maximum # of significant digits *
116 !* OR binary precision if < 0 *
117 !* *
118 !* Output parameters: *
119 !* NBITS INTEGER Number of bits for packing *
120 !* ISCALE INTEGER Power of 10 scaling to use *
121 !* IRET INTEGER Return code *
122 !* 0 = normal return *
123 !** *
124 !* Log: *
125 !* K. Brill/NMC 06/92 *
